## 1. 프로젝트 개요

### 1-1. 목적
경옥채의 다채널(한약국 매장 · 백화점 입점 · 자사몰 · 이벤트) 운영을 단일 시스템으로 통합하여 재고·판매·고객·생산·회계·배송을 일원화하고, AI 에이전트 기반 업무 자동화로 인력 대비 처리량을 3~5배 향상시키는 것을 목표로 함.

### 1-2. 개발 범위 (A-급 완성도 기준)
- 웹 기반 ERP/CRM/POS 통합 시스템 (20+ 화면)
- RBAC 기반 5개 역할 권한 분리
- 외부 시스템 연동 3종 (Cafe24, SweetTracker, Solapi)
- AI 에이전트 (Function Calling, 49개 도구)
- 회계 자동 분개 + 배송 관리 + 재고 실사 확장 기능

### 1-3. 기술 스택
| 계층 | 기술 |
|---|---|
| 프론트엔드 | Next.js 16 (App Router), React, TypeScript, Tailwind CSS v4 |
| 백엔드 | Next.js Server Actions, API Routes |
| 데이터베이스 | Supabase (PostgreSQL), RLS 정책 |
| 인증 | Custom Session (SHA-256, httpOnly cookies) |
| AI | Claude API (Anthropic) — Function Calling |
| 외부 연동 | Cafe24 OAuth/Webhook, SweetTracker, Solapi, 대한통운 엑셀 |
| 배포 | Vercel (프론트/서버), Supabase (DB/Storage) |
